# Content System — одна идея в медиамашину

Одностраничное демо на **Next.js + Tailwind CSS**. Превращает одну сырую идею
в полноценную контент-систему за несколько секунд — без бэкенда, базы и API.
Вся генерация клиентская (mock на основе введённого текста).

## Что делает

1. Пользователь вводит идею и жмёт **«Развернуть в контент-систему»**.
2. Идея проходит анимированный пайплайн: **Идея → Нарратив → Форматы → Дистрибуция**.
3. Карточки появляются одна за другой:
   - **YouTube** — структура видео (тайм-коды + сценарные блоки);
   - **Telegram** — готовый пост;
   - **Reels / Shorts** — 5 хуков на первые секунды.
4. У каждой карточки: название формата, сгенерированный текст, кнопка копирования,
   **leverage score** (кольцо) и плавная glow-анимация.

## Стек

- Next.js 14 (App Router), React 18
- Tailwind CSS 3
- Никакого бэкенда / API — `lib/generate.js` собирает контент детерминированно
  из текста пользователя.

## Запуск

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # прод-сборка
```

## Структура

```
new app/
├── app/
│   ├── layout.jsx       # корневой layout + метаданные
│   ├── page.jsx         # экран: ввод → генерация → результат
│   └── globals.css      # фон-сетка, glow-рамки, keyframes
├── components/
│   ├── Pipeline.jsx     # визуальный пайплайн с подсветкой фаз
│   └── ContentCard.jsx  # карточка формата + копирование + leverage-кольцо
├── lib/
│   └── generate.js      # клиентская mock-генерация контент-системы
└── tailwind.config.js   # палитра (графит/плазма/электрик) и анимации
```
