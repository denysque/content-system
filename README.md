# Content System — AI-помощник ведущего

Веб-приложение: вводишь **тему мероприятия** — AI (Claude Sonnet 4.6) разворачивает
её в **6 готовых материалов для ведущего** за несколько секунд.

🔗 Прод: https://content-system.vercel.app

## Что делает

1. Пользователь входит по ссылке на почту (magic link) и вводит тему мероприятия
   (например: «свадьба в стиле Великого Гэтсби», «корпоратив IT-компании»).
2. Тема проходит анимированный пайплайн: **Идея → Сценарий → Форматы → Материалы**.
3. Появляются 6 карточек, сгенерированных Claude:
   - **Текстовый сценарий ведущего** — живой текст открытия от первого лица;
   - **Визуальные идеи и концепция** — оформление, палитра, фотозона, дресс-код;
   - **Сценарный план** — тайминг вечера по блокам;
   - **Банк шуток и подводок** — уместный юмор под тему;
   - **Чек-лист подготовки** — что подготовить заранее;
   - **Список идей и активностей** — конкурсы и интерактивы.
4. У каждой карточки: название, текст, кнопка копирования, **leverage-кольцо**, glow-анимация.
5. Наборы можно **сохранять** (Supabase) и возвращаться к ним.

## Стек

- **Next.js 14** (App Router), **React 18**
- **Tailwind CSS 3** (тёмная тема, акценты plasma/electric)
- **Anthropic Claude Sonnet 4.6** — генерация (серверный route `/api/generate`)
- **Supabase** — авторизация magic link + хранение (таблица `event_systems`, RLS)
- **Vercel** — хостинг + Web Analytics

## Запуск локально

```bash
npm install
# создать .env.local с ключами (см. ниже и SUPABASE_SETUP.md)
npm run dev      # http://localhost:3000
npm run build    # прод-сборка
```

### Переменные окружения (`.env.local`)

| Переменная | Назначение |
|---|---|
| `ANTHROPIC_API_KEY` | серверный ключ Claude (НЕ `NEXT_PUBLIC`) |
| `NEXT_PUBLIC_SUPABASE_URL` | URL проекта Supabase |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | publishable-ключ Supabase |

Подробная настройка backend (таблица, RLS, redirect URLs) — в [`SUPABASE_SETUP.md`](./SUPABASE_SETUP.md).

## Деплой

```bash
vercel --prod --yes     # ручной деплой (авто-деплой из GitHub не подключён)
```
